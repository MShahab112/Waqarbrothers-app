# 🚀 Waqar Brothers Digital Account Book

A simple and powerful desktop application to manage daily recovery and expenses for business operations.

---

## ✨ Features

* 📥 Add recovery and expense entries
* 📅 Navigate daily records
* 📊 Reports with charts and insights
* 📄 Professional PDF reports
* 🔐 Secure login system
* 💾 Backup & restore system
* ⚡ Fast and optimized performance

---

## 📥 Installation

1. Download the latest version from Releases
2. Run `WaqarBrothersSetup.exe`
3. Install the application
4. Launch from Desktop shortcut

---

## 🔐 Default Login

* Username: `admin`
* Password: `admin123`

⚠️ You will be asked to change the password on first login.

---

## 📊 Usage

* Add daily recovery and expense entries
* Close the day after completing entries
* View reports and insights
* Take regular backups

---

## 💾 Data Location

All data is stored locally on your system.

⚠️ Do not delete database files manually.

---

## 🛠️ Tech Stack

* Python (Flask)
* SQLite
* HTML / CSS / JavaScript
* Chart.js
